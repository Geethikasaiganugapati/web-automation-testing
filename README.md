# Selenium Java Automation Framework

## Overview

This project is a Selenium Automation Testing Framework developed using Java, Selenium WebDriver, TestNG, and Maven.

The framework automates login and user workflows using the Page Object Model (POM) design pattern.

## Features

- Automated Login Testing
- Selenium WebDriver Integration
- TestNG Assertions and Reporting
- Page Object Model (POM)
- Screenshot Capture
- Regression Testing
- Maven Dependency Management

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- WebDriverManager
